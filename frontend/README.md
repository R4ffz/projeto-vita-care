# Frontend — VitaCare IoT

Aplicação React + Vite + TypeScript responsável pela interface do sistema.

Telas previstas:

- Login.
- Central multi-paciente (grid de cards ordenados por gravidade).
- Dashboard do paciente (sinais vitais em tempo real).
- Histórico gráfico por paciente.
- Lista de alertas e modal de emergência para quedas.
- Cadastro de paciente.
- Painel de controle do simulador.

Bibliotecas principais: Tailwind CSS, Axios, `@stomp/stompjs` + SockJS, Chart.js.

Conteúdo será populado a partir da Fase 8 do plano de implementação.
